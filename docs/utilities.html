Utilities¶

In general, functions listed below will not modify the input objects and return a copy instead, while methods for pypianoroll.Track and pypianoroll.Multitrack may modify the input objects.

pypianoroll.check_pianoroll(arr)[source]¶

Return True if the array is a standard piano-roll matrix. Otherwise, return False. Raise TypeError if the input object is not a numpy array.

pypianoroll.assign_constant(obj, value)[source]¶

Assign a constant value to the nonzeros in the piano-roll(s). If a piano-roll is not binarized, its data type will be preserved. If a piano-roll is binarized, it will be casted to the type of value.

Parameters:value (int or float) – The constant value to be assigned to the nonzeros of the piano-roll(s).
pypianoroll.binarize(obj, threshold=0)[source]¶

Return a copy of the object with binarized piano-roll(s).

Parameters:threshold (int or float) – Threshold to binarize the piano-roll(s). Default to zero.
pypianoroll.clip(obj, lower=0, upper=127)[source]¶

Return a copy of the object with piano-roll(s) clipped by a lower bound and an upper bound specified by lower and upper, respectively.

Parameters:
  • lower (int or float) – The lower bound to clip the piano-roll. Default to 0.
  • upper (int or float) – The upper bound to clip the piano-roll. Default to 127.
pypianoroll.copy(obj)[source]¶

Return a copy of the object.

pypianoroll.pad(obj, pad_length)[source]¶

Return a copy of the object with piano-roll padded with zeros at the end along the time axis.

Parameters:pad_length (int) – The length to pad along the time axis with zeros.
pypianoroll.pad_to_multiple(obj, factor)[source]¶

Return a copy of the object with its piano-roll padded with zeros at the end along the time axis with the minimal length that make the length of the resulting piano-roll a multiple of factor.

Parameters:factor (int) – The value which the length of the resulting piano-roll will be a multiple of.
pypianoroll.pad_to_same(obj)[source]¶

Return a copy of the object with shorter piano-rolls padded with zeros at the end along the time axis to the length of the piano-roll with the maximal length.

pypianoroll.transpose(obj, semitone)[source]¶

Return a copy of the object with piano-roll(s) transposed by semitones semitones.

Parameters:semitone (int) – Number of semitones to transpose the piano-roll(s).
pypianoroll.trim_trailing_silence(obj)[source]¶

Return a copy of the object with trimmed trailing silence of the piano-roll(s).
